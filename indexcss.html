üîê Access System (/access)

C3B1XBOT menggunakan sistem akses per-command. Secara default, semua command bisa digunakan semua user (selama permission Discord mencukupi), kecuali jika di-lock melalui /access.

1. Owner Only

Command tertentu bisa dibuat hanya untuk owner bot. 

/access add-command-owner command: access, c3b1xbot-update

Contoh command owner:

  • /access
  • /c3b1xbot-update
  • /update-log
  • /setup-config

2. User Whitelist (Premium)

Beri akses ke user tertentu untuk command premium tanpa mengganggu command lain. 

/access add-command-user
  command: stock, send-product
  user: @UserPremium

Cocok untuk:

  • Premium store manager
  • Staff khusus

3. Role Whitelist (Admin)

Beri akses command ke role tertentu per server. 

/access add-command-role
  command: ban, kick, timeout
  role: @Admin

Contoh: hanya role @Admin yang bisa pakai command moderasi tertentu.

Owner bot selalu bisa memakai semua command, walaupun command tersebut di-lock untuk user/role lain.
